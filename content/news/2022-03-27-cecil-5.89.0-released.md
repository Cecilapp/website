### Features

#### WebP image optimization

[WebP](https://developers.google.com/speed/webp) images can be optimized (compressed) by enabling the following option in config file:

```yaml
assets:
  images:
    optimize:
      enabled: true
```

It works in templates with the [`asset()` function](https://cecil.app/documentation/templates/#asset) and in [pagesâ€™ body](https://cecil.app/documentation/content#body) with image Markdown tag.
