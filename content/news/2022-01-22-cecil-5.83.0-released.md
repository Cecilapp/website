### Features

#### Use a custom image variable in metatags

By default the _[metatags](https://cecil.app/documentation/configuration#metatags)_ partial template uses the `image` variable of the current page, or the global `site.image` if the page doesnâ€™t have an image.

So, if you want to use a custom image variable, you can do it by adding the following code in the template:

```twig
{% set image = page.other_image_variable %}
{% include 'partials/metatags.html.twig' %}
```
