### Fonctionnalités

#### Supprimer les fichiers de cache par expression régulière

*Exemple:*

```bash
php cecil.phar serve --clear-cache="\.css"
```

> Supprime les fichiers CSS mis en cache et préserve le reste du cache.

[Documentation →](/documentation/commandes/#serve)
