Depuis la création de Cecil, le seul moyen de connaître les nouvelles fonctionnalités et autres correctifs introduits dans une nouvelle version était la [page Releases de GitHub](https://github.com/Cecilapp/Cecil/releases) du projet.

Néanmoins, les informations communiquées restaient très techniques et pas toujours détaillées (à moins de mettre le nez dans le code lui-même).

Aussi, nous avons créé la **[section Actualités](/news/)** afin de pouvoir communiquer plus explicitement avec les utilisateurs sur :

1. Quelles sont les nouveautés d’une version majeure ?
2. Quels bugs sont corrigés dans une version mineure ?

Nous espérons qu'il vous plaira 🤵
