### Fonctionnalités

#### Markdown « insérer » en ligne

_Exemple d'utilisation_ :

```yaml
++text++
```

est converti en :

```html
<ins>text</ins>
```

### Correctifs

- Les notes de bas de page sont désormais supprimées dans l'extrait
- Vérifie si le fichier de configuration existe/est défini
- La variable `$layout` pourrait ne pas être définie
- La fonction `str_replace` PHP est corrigée pour PHP 8
- `bindtextdomain` est défini uniquement si le `locale` (l10n) existe
- Optimisation des images d'actifs corrigée
- Extension `.ser` ajoutée aux fichiers cache
- Processus de redimensionnement des actifs corrigé
- Les actifs sont optimisés sur `save()`

### Divers

- Dépendances mises à jour
- Graphiques Docs mis à jour
- `declare(strict_types=1)` ajouté à tous les fichiers de bibliothèque
- Code nettoyé
