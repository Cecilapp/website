### Fonctionnalités

#### Nouvelle variable `canonical` pour remplacer l'URL canonique générée

Si le `metatags.html.twig` partiel est inclus dans votre modèle principal, une URL canonique est automatiquement ajoutée à la page `<head>`.  
Avec la variable `canonical` (définie dans la page [front Matter](/documentation/content/#front-matter)), vous pouvez remplacer l'URL canonique et le titre canonique :

```yaml
canonical:
  url: <URL>
  title: "<URL title>" # optional
```
