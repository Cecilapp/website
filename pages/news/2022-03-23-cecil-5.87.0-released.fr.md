### Fonctionnalités

#### Élément de niveau bloc Markdown _Note_

Un nouvel élément Markdown est disponible : **_Note_**.

Il encapsule un texte Markdown (Information, conseil, important, etc. Ce dont vous avez besoin) dans un élément `<aside>` avec une classe CSS dédiée, nommée selon le type de la note, c'est-à-dire `class="note note-<type>"`.

```markdown
:::tip
**Tip:** This is an advice.
:::
```

Est converti en :

```html
<aside class="note note-tip">
  <p>
    <strong>Tip:</strong> This is an advice.
  </p>
</aside>
```

> Dans l'exemple précédent `tip` est le type de la note.

Exemples dans le contexte de ce site Web :

:::astuce
**Conseil :** Ceci est un conseil.
:::

:::infos
**Info :** Ceci est une note d'information.
:::

:::important
**Important :** Il s'agit d'un message important.
:::

:::not_styled
Ceci est une note.
:::
