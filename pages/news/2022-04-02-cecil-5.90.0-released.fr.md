### Fonctionnalités

#### Option pour désactiver la récupération d'images à distance

Avant la version 5.90, les images distantes (c'est-à-dire externes) utilisées dans le corps des pages (ex : `![](https://example.com/image.jpg)`) étaient systématiquement récupérées et stockées localement, dans le répertoire `_sites/assets/`.  
Cependant, ce n’est pas un comportement que les utilisateurs de Cecil souhaitent systématiquement, c’est pourquoi nous avons ajouté une option pour le désactiver :

```yaml
body:
  images:
    remote:
      enabled: false
```

:::infos
**Info :** Si vous désactivez l'option _remote_ Cecil ne sait pas gérer les fichiers images (pas de redimensionnement, pas d'optimisation, etc.) : l'URL sera utilisée telle quelle dans la balise `<img>`.
:::
