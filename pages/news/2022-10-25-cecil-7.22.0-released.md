### Features

#### Support of embed GitHub Gist link

GitHub Gist can be easly embed by adding the `{embed}` attribute.

```markdown
[Cecil build script](https://gist.github.com/ArnaudLigny/6b2aa9e6b25581c96435e9296efe0c0e){embed}
```
