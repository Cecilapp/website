### Changements clés

1. simplification de la structure de configuration
2. Amélioration du cache : performances légèrement inférieures, mais confort accru
3. meilleure gestion des messages d'erreur

### Améliorations

#### Configuration

- Une option peut être désactivée directement avec `option:false` ou avec la sous-touche `enabled` : `option:enabled:false`
- La configuration des thèmes est automatiquement importée sur `setConfig()` et `setSourceDir()`
- Les options obsolètes sont désormais répertoriées dans le fichier dédié `config/deprecated.php`

#### Cache et ressources

- Mécanismes de cache reconstruits : légèrement moins performants, ils gèrent mieux les modifications de fichiers et actualisent automatiquement le cache
- Les fichiers distants sont désormais mis en cache pendant 7 jours

### Changements majeurs

- Les taxonomies par défaut `tag` ont `category` été supprimées. Si vous les avez utilisées, vous devez les ajouter une par une à la configuration de votre site.
- Options de configuration déplacées :
  - `pagination` -> `pages:pagination`
  - `paths` -> `pages:paths`
- Options d'actifs renommées :
  - `remote_fallback` -> `fallback`
  - `force_slash` -> `leading_slash`
