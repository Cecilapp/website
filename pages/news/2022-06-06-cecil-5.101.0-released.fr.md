### Fonctionnalités

#### Mise en évidence de la syntaxe du code dans Markdown

La surbrillance de la syntaxe du code est désormais prise en charge nativement dans [Markdown](/documentation/content/#syntax-highlight), *highlight.js* n'est plus requis dans le modèle.

_Exemple:_

```php
<?php
// PHP code here
echo "test";
```

### Correctifs

- Le délai d'attente du service est désormais fixé à 2 x 60 secondes
