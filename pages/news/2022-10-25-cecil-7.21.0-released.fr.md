### Fonctionnalités

#### Solution de secours locale pour les images distantes

Vous pouvez spécifier une image de secours – générique – au cas où les images distantes ne seraient pas disponibles.

```yaml
body:
  images:
    remote:
      enabled: true
      fallback:
        enabled: true # enables a fallback if remote image is not found (false by default)
        path: 'images/no-image.png' # path to the fallback image, stored in `assets` dir (empty by default)
```
