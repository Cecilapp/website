### Features

#### Open `content` with the configured editor

Uses the command option `--open` (`-o`) to open the content directory with the configured editor.

_Example:_

```bash
cecil --open
```

Defines the editor with the key `editor`.

_Example:_

```yaml
editor: typora
```
