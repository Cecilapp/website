### Fonctionnalités

#### Utiliser une variable d'image personnalisée dans les balises méta

Par défaut, le modèle partiel _[metatags](/documentation/configuration#metatags)_ utilise la variable `image` de la page actuelle, ou le `site.image` global si la page n'a pas d'image.

Ainsi, si vous souhaitez utiliser une variable d'image personnalisée, vous pouvez le faire en ajoutant le code suivant dans le modèle :

```twig
{% set image = page.other_image_variable %}
{% include 'partials/metatags.html.twig' %}
```
